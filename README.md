**Лабораторные работы 2 семестр**

**Lab 1: Поиск последовательных нулей
**
Задача: Определить, имеется ли в массиве два идущих подряд нулевых элемента.

**Lab 2: Подсчёт уникальных чисел
**
Задача: Подсчитать количество различных чисел в числовом массиве.

**Lab 3: Работа с матрицей и вектором
**
Задача: Даны вектор x и квадратная матрица A. Заменить все чётные строки матрицы на вектор x.

**Lab 4: Обработка строк
**
Задача: Дан массив из n строк (описан как указатель на указатель). Удалить все строки, оканчивающиеся на "ing" или "ed".

**Lab 5: Деление на 2
**
Задача: Реализация деления числа на 2 с использованием битовых операций.

**Lab 6: Поиск подстроки
**
Задача: Линейный поиск подстроки в строке.

**Lab 7: Ханойские башни
**
Задача: Реализация решения головоломки "Ханойские башни" с рекурсивным алгоритмом.

**Lab 8: Польская инверсная запись
**
Задача: Реализация алгоритма вычисления выражений в польской инверсной записи (RPN).

**Lab 9: Булевы векторы
**
Задача: Даны два булевых вектора. Определить, являются ли они сравнимыми (каждый элемент одного >= соответствующему элементу другого). Если возможно, упорядочить их.

**Lab 10: Работа со списком
**
Задача: Переместить элемент двусвязного списка на k позиций вперёд (если элементов меньше k — переместить в конец). Вывести указатели на первый и последний элементы. Без использования выделения/освобождения памяти.
